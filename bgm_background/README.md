# bgm_background/

背景に使う動画（実写の雨・暖炉・宇宙など）をこのフォルダに入れてください。
動画があればそれが1時間ループの背景になり、なければAI画像/グラデーション背景になります。

## 無料で商用利用OKの動画の入手先（クレジット表記不要）

- Pexels: https://www.pexels.com/search/videos/rain%20window%20night/
- Pixabay: https://pixabay.com/videos/search/rain%20window/

「rain window night」「rain glass loop」などで検索し、暗め・動きが穏やかなものを選ぶと睡眠用BGMに合います。

## ルール

- 対応形式: mp4 / mov / webm / mkv / avi
- ニッチ名と同じファイル名（例: `rain_sleep.mp4`）があれば優先的に使われます
- GitHubのWeb画面からアップロードする場合は **1ファイル25MBまで**。
  ダウンロード時に小さめの解像度（HD/SD）を選ぶか、20秒程度の短いループを選んでください
- 「seamless loop（継ぎ目のないループ）」と書かれた動画を選ぶと、繰り返しが目立ちません
