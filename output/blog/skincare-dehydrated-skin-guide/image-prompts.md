# 画像生成プロンプト集

ブログタイトル: **混合肌・インナードライ肌の乾燥対策｜うるおいを守るスキンケア方法を解説**

このファイルは、ChatGPT（GPT-4o / DALL·E / GPT Image）でブログ用の画像を生成するためのプロンプト集です。
そのままチャットに貼り付けて使えるよう、日本語プロンプトと英語プロンプトの両方を用意しています。

## 共通スタイルガイド

- 全体トーン: 清潔感、透明感、ナチュラル、淡いパステル（ベージュ／オフホワイト／ライトピンク／ミントグリーン）
- 写真風: 自然光、柔らかい陰影、被写界深度浅め、ミニマル
- イラスト風: フラットデザイン、線が細く柔らかい、肌色は中立的（特定の人種に偏らない）
- 文字: 画像内に日本語の文字を入れない（後からブログ側で重ねる）
- 禁止要素: 過度な美白表現、特定のブランドロゴ、医療的に断定的な表現を示唆するアイコン

## 推奨アスペクト比

| 用途 | アスペクト比 | 想定サイズ |
| --- | --- | --- |
| アイキャッチ（ヒーロー） | 16:9 | 1792×1024 |
| 本文中セクション挿絵 | 4:3 または 16:9 | 1024×768 / 1792×1024 |
| SNSシェア（Instagram等） | 1:1 | 1024×1024 |
| 縦長（Pinterest / ストーリー） | 9:16 | 1024×1792 |

## ChatGPTでの使い方

1. ChatGPT（GPT-4o もしくは画像生成対応モデル）を開く。
2. 上記の日本語プロンプトをそのまま貼り付け、最後に「この内容で画像を生成してください。アスペクト比は◯◯にしてください」と指示。
3. 生成された画像が好みでない場合、「もっと淡い色で」「人物は入れずに」など追加の指示で微調整。
4. 商用利用する場合は、生成画像のライセンスをOpenAIの利用規約で必ず確認すること。

## 注意事項

- 医療・薬機法に関わる表現（治る・治療・効果効能の断定など）を画像内に入れないように注意。
- 特定ブランドや既存パッケージを模した画像は避ける（ラベルはぼかすか、ラベルなしにする）。
- モデル人物を使用する場合、特定の有名人に似た顔にしない。
